# to-do-list
A simple and interactive "to-do list" using HTML, CSS and JavaScript. <i>New update</i>: Cookies enabled! Your data is now saved safely in your browser.

### Features
* Add new task
* Mark task as completed / uncompleted
* Delete single task
* Delete all tasks
* Update username

### Technologies used:
* HTML
* CSS
* JavaScript
* SweetAlert2
